# RMIT Connect - Student Connection Platform

A clean, professional student connection platform built with Flask and styled with RMIT branding.

## 🚀 Quick Start

```bash
cd /Users/daniloslavniy/Desktop/UserAuthSystem
/Users/daniloslavniy/Desktop/UserAuthSystem/env/bin/python main.py
```

**App will be available at:** http://localhost:5001

## 📁 Clean Project Structure

```
UserAuthSystem/
├── main.py              # Application entry point
├── app.py               # Flask app configuration
├── routes.py            # All application routes
├── models.py            # Database models
├── static/
```

## ✨ Key Features

- **RMIT Branding**: Official colors and typography
- **Clean History**: 3 focused tabs (Likes Received, Connections, My Likes)
- **Profile Viewing**: Clean profile view without action buttons
- **Mutual Matching**: Smart connection detection
- **Responsive Design**: Works on all devices

## 🎯 Navigation

1. **Likes Received** - People who liked you (primary tab)
2. **Connections** - Mutual matches and established connections
3. **My Likes** - People you have liked

## 🔧 Technical Stack

- **Backend**: Flask + SQLAlchemy
- **Frontend**: Bootstrap 5 + RMIT Custom CSS
- **Database**: SQLite
- **Authentication**: Session-based with Flask-Login
